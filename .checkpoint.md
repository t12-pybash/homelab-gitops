## Flux Bootstrap Process

1. Terraform applies flux_bootstrap_git resource
2. Creates FluxSystemNamespace, flux controllers
3. Creates GitRepository pointing to this repo
4. Creates root Kustomization for infrastructure/apps/monitoring
5. Flux auto-reconciles every 1-5 minutes
